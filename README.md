# LaTeX CV

Minimalist CV template using moderncv, optimised for research positions and ATS compatibility.

## Compile

```bash
pdflatex "Fabio_J_Fehr_CV_2025.tex"
```

Or use **Ctrl+Alt+B** in VS Code with LaTeX Workshop.

## Edit

- `Fabio_J_Fehr_CV_2025.tex` - Main template
- `cv-content.tex` - Personal content
